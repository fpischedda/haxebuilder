HaxeBuilder the ultimate automated Haxe building system

With HaxeBuilder you can:
  • Monitor you repository and get a new build as soon as you push changes to the tracked branch
  • Inspect you past builds, download build artifacts such as ready to deploy HTML5 code, Executables, APKs etc
  • Test your game directly from your dashboard (HTML5 builds only)
